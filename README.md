# Maverick Administrators Brown & Brown Portal

Static single-file HTML Brown & Brown portal for surrogacy agencies to look up their agency code, review state-specific singleton policy rates, and open the correct HubSpot application form.

## Hosting

This repository is intended for GitHub Pages. The site is served from `index.html` at the repository root.

## Maintenance

Agency data, rate tables, and HubSpot form URLs are embedded in the `<script>` section of `index.html`.
